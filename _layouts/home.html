Story

I started out programming 2D games in Game Maker 6. I had a goal to write the first stable MMORPG using the Game Maker toolset. I wrote a 14 page game design document that I stuck to while developing over 50,000 lines of code on a live stream with viewers hacking, overloading and botting in real time. Since then I began working with HTML5 and Javascript to work on countless other projects. I have written web apps for eCommerce and restaurants to one of the fastest web sockets engines in node.js. I enjoy the challenge of real time network programming.
